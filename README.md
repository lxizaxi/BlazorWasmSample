# 目的
- Blazorのお勉強

# 使い方メモ
## wasmのprojectの作成
```
$ dotnet new blazorwasm -o (プロジェクト名)
```
## app起動方法
作成したプロジェクト直下で、以下のコマンド
```
$ dotnet run
```
